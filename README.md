## Featured Projects

### [FlowForge](https://github.com/igalVilensky/flowforge)

A safety and observability layer for AI automation prototypes. FlowForge turns a plain-language automation idea into a structured, human-gated blueprint before anything is connected to production tools.

**Focus:** AI automation safety, workflow compilation, human approval gates, agent trace visibility  
**Tech:** Nuxt, Vue, TypeScript, Nuxt server routes, Zod, Groq, Gemini fallback  
**Highlights:**

- Guided clarification for vague automation requests
- Compile modes for rule-only, balanced, and full agent-assisted flows
- Safety review with blocked actions, risk outcomes, and approval gates
- Agent trace/debug visibility and provider fallback handling
- n8n implementation handoff prompt and draft workflow JSON generation

---

### [Job Command Center](https://github.com/igalVilensky/job-command-center)

A self-hosted AI-assisted job search operating system designed for structured application tracking, decision support, and safe human-in-the-loop workflows.

**Focus:** AI automation, job workflow management, self-hosted product design  
**Tech:** TypeScript, full-stack architecture, AI provider integration, Docker-oriented local setup  
**Highlights:**

- Self-hosted-first approach with user-owned data and provider keys
- Optional AI providers instead of mandatory external dependencies
- Mock AI mode for local development without API keys
- Designed as both a practical tool and a portfolio-grade AI product system

### [Decision Matrix AI](https://github.com/igalVilensky/decision-matrix-ai)

AI-assisted weighted decision matrix for comparing complex options such as tools, products, jobs, courses, vendors, business ideas, or other tradeoff-heavy choices.

**Focus:** AI-assisted decision support, explainable scoring, structured product UX  
**Tech:** React, Vite, TypeScript, Tailwind CSS, Netlify Functions, Groq API, Firebase Auth, Firestore, Zod  
**Highlights:**

- Weighted criteria, must-have rules, manual scoring, and AI-suggested scores
- Ranked results, category breakdowns, strengths, weaknesses, and explainable recommendations
- Human review flow for accepting, editing, or rejecting AI suggestions
- Firestore-backed saved matrices per user
- JSON import/export and action checklist generation
